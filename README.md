# nanoc-sass

This provides a [Sass](http://sass-lang.com/) filter for [nanoc](http://nanoc.ws).

## Installation

`gem install nanoc-sass`

## Usage

```ruby
filter :sass
```

Options passed to this filter will be passed on to Sass.
